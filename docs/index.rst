iitp_6_python: Hough Lines Detection
=====================================

.. toctree::
   :hidden:
   :maxdepth: 1

The example project for implementing Hough transform for detecting lines.


Installation
------------

To install the iitp_6_python project, run this command in your terminal:

.. code-block:: console

   $ pip install iitp_6_python


Usage
-----

iitp_6_python's usage looks like:

.. code-block:: console

   $ iitp_6_python [OPTIONS]

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
